# Transfer Learning Food Vision

Jupyter notebook: `transfer_learning_food_vision.ipynb`

Description:
- Transfer learning example for a food classification task (pretrained backbone + custom head).

How to run:
1. Open the notebook in Jupyter, JupyterLab, or Google Colab.
2. Install dependencies (example):

```
pip install tensorflow numpy matplotlib pillow
```
