# Machine Learning Projects

A collection of classic ML projects and notebooks covering supervised/unsupervised learning, model selection, and evaluation.

## Topics
- Classification: Logistic Regression, SVM, k-NN, Tree/Forest, XGBoost
- Regression: Linear/Ridge/Lasso, SVR
- Unsupervised: K-Means, PCA, clustering
- Model selection: cross‑validation, metrics, confusion matrix, ROC/AUC

## Structure
- notebooks/ – Jupyter notebooks per topic
- data/ – Sample datasets or download scripts
- src/ – Helpers for preprocessing, training, and evaluation

## Quickstart
```bash
# (Optional) create and activate a virtual environment
python -m venv venv
./venv/Scripts/activate  # Windows
# Install core deps
pip install numpy pandas scikit-learn matplotlib seaborn
# Launch Jupyter
pip install jupyter
jupyter notebook
```
